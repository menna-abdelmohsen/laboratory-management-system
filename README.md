# Laboratory Management System

A database-driven Laboratory Management System built to manage patients, laboratory tests, samples, reports, and workflow status in an organized and efficient way.

## Overview

This project is designed to help laboratories manage daily operations such as patient registration, test selection, sample tracking, report generation, and data storage.
It focuses on improving accuracy, organization, and accessibility inside laboratory workflows.

## Features

* Patient registration and data management
* Test selection from predefined laboratory tests
* Sample tracking with status management
* Report generation workflow
* Database storage using SQL
* User-friendly interface
* Organized records for patients, tests, and reports

## Technologies Used

* Python
* SQL
* Streamlit
* Database Management
* File Handling
* User Interface Design

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/menna-abdelmohsen/laboratory-management-system.git
```

2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Run the project:

```bash
streamlit run app.py
```

## Notes

This project uses sample/test data only.
No real patient data is included.
